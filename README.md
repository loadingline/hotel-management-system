# Hotel Management System

## Software Modeling and Design Project  
**Course:** SWE202 – Software Modeling and Design  
**Project Type:** System Analysis and System Modeling  

---

## 1. Project Description
The Hotel Management System is a conceptual software system designed to automate and manage core hotel operations. The system focuses on structured analysis, requirement specification, and UML-based system modeling.
This project emphasizes proper software engineering practices including requirement formalization, actor identification, and system decomposition.
The system supports operational, administrative, and managerial processes within a hotel environment.

---

## 2. Objectives
The main objectives of this project are:
- To define clear and structured system requirements  
- To model system behavior using UML diagrams  
- To design a role-based hotel management platform  
- To demonstrate application of software modeling principles  
- To ensure traceability between requirements and system models  

---

## 3. System Scope

The system includes the following core modules:
- User Authentication and Role Management  
- Room Management  
- Reservation Management  
- Check-in / Check-out Processing  
- Payment Management  
- Reporting and Statistics  
The system does not include external integrations (e.g., third-party booking platforms) in its initial scope.

---

## 4. Actors

The system defines four primary actors:
- Guest  
- Receptionist  
- Hotel Manager  
- System Administrator  
Each actor interacts with the system based on assigned permissions and responsibilities.
